uso de <iframe>

La etiqueta <iframe> en HTML5 se utiliza para incrustar contenido externo dentro de una página web. Esto puede incluir mapas, videos, otras páginas web, entre otros. Con los atributos width, height y allowfullscreen, se puede controlar el tamaño y la capacidad de visualización del contenido incrustado.
