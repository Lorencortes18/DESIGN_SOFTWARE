ETIQUETAS HTML

<div>
La etiqueta <div> es un contenedor genérico que agrupa elementos para aplicarles estilos o manipulaciones mediante JavaScript. Documentación
<p>
La etiqueta <p> El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.Documentación
<img>
La etiqueta <img> para incluir imágenes en el contenido de una página utilizaremos la etiqueta, Requiere el atributo src para especificar la fuente de la imagen. Documentación
<h1>
Las etiquetas <h1> H1 representa el encabezado más importante de un documento. Los niveles de encabezados se utilizan para estructurar y organizar el contenido de una página web en diferentes niveles de importancia y jerarquía.Documentación
<h6>
La etiqueta <h6> el H6 se reserva para los títulos menos relevantes en una página web. Su uso excesivo puede resultar contraproducente, diluyendo la jerarquía de la información y confundiendo al lector. Por lo tanto, es crucial utilizar el H6 con moderación y precisión.Documentación
<ul>
La etiqueta <ul> se usa para definir una lista no ordenada, generalmente acompañada de <li> para los elementos de la lista.Documentación
<ol>
La etiqueta <ol> El elemento ol permite definir listas o viñetas ordenadas ("Ordered List"), bien con numeración o alfabéticamente.Documentación
<table>
La etiqueta <table> representa datos en dos o mas dimensiones, que puede contener otras etiquetas como <tr> (para las filas), <td> (para las celdas), y <th> (para las cabeceras de las columnas).Documentación
<form>
La etiqueta <form> representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web.Documentación
<strong>
La etiqueta <strong> se utiliza para resaltar texto, indicando que es importante, usualmente se muestra en negrita. Documentación
